## 🔐 Authentication Notes

### How It Works (Demo)
- No backend required
- Instant login without email verification
- Users stored in localStorage
- Demo account: username "demo", any password
- Auto-generates profile & avatar

### For Production
When building for real users:
1. Integrate with real auth provider (Supabase, Firebase, Auth0)
2. Add password hashing (bcrypt)
3. Implement email verification
4. Add password reset flow
5. Use secure session management
6. Add rate limiting
