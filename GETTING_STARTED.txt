❓ FAQ:

Q: Do I need to verify my email?
A: No! Login with any username and password instantly.

Q: Is this a real backend?
A: No, it uses localStorage for demo. For production, add a real
   backend (Supabase, Firebase, etc.)

Q: Can I customize the design?
A: Yes! Edit tailwind.config.ts and globals.css

Q: How do I deploy?
A: Follow DEPLOYMENT_GUIDE.md for step-by-step instructions

Q: Can I add my own images?
A: Yes! Update SAMPLE_IMAGES in new-post-dialog.tsx

Q: Does it work offline?
A: Yes! Everything runs in the browser (localStorage)

Q: Can I add more features?
A: Absolutely! The code is yours to customize
